Ribbon Strat

Midi Instrument for Software Synthesizers
Elective | Physical Interaction

usbMIDI(RibbonStrat);

Die RibbonStrat ist ein Projekt vom Physical Interaction Kurs aus dem Sommer Semester 2016. Sie hat anschlagsdynamische "Pads"(FSRs), womit Töne angetriggert werden. Wird die "Saite"(SoftPotentiometer) gedrückt, wird abhängig von der Position die entsprechend gepitchte Note gespielt. Ähnlich wie bei einer Gitarre. Mit den Pushbuttons kann man das Tuning der Pads variieren. Der Joystick kann auf jeden beliebigen Poti in der Synthie-Software per MIDI-Learn gemappt werden.



Betreunde Dozenten: Claudius Coenen, Michael Allinger
