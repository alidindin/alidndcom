The Growing

Die Partyreihe the Growing hat digitale Kunst auf ihre Fahnen geschrieben. Mitbegründer David Bueno gibt so Künstlern und Nerds die Bühne, Exponante auf Electro Beats auszustellen.
Auf mehreren Veranstaltungen haben wir eine simple Installation aufgestellt. Ein Kurzdistanzbeamer bestrahlt eine helle Wand. Ein Beispielpatch aus der dx11 Pointcloud brauchte nur angepasst werden. Die Kinect trackt Personen, umhüllt sie mit Partikeln die mit vereinzelten Kräften umhergeschleudert werden. Der User kann mit Lichtreflexionen und Körperbewegungen Einfluss auf die Partikel nehmen.
